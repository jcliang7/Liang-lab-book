# Vue-Setup

## node.js  用 16.15.1

切換使用的node版本（小提示，可以用tab補齊版本號）

```
nvm use v15.16.1
```

查看已安裝的node版本

```
node -v
```

查看目前使用的版本

```
nvm current
```
